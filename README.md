h1>Fruits-Bats</h1>

Fruits and Bats is a side scrolling, jump and run game built with Javascript and the P5 library.




Collect as much fruit as you can within the time limit and avoid the fire and bats!


** Controls **

A ---> Move Left
D ---> Move Right
Space ---> Jump
